# Backend

Contenitore della logica di supporto.

## Futuri moduli

- integrazione API TCG
- normalizzazione dati carta
- aggiornamento quotazioni
- motore analisi mazzi
- validazione regole formato

## Nota

Il backend può essere leggero o addirittura assente se tutta la logica viene mantenuta nel client, ma è utile separare almeno le responsabilità di sincronizzazione e calcolo.
